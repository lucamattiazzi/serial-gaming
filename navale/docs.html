Battaglia navale — API del bot

Il primo gioco a informazione nascosta: l'arbitro custodisce le flotte, tu vedi solo i tuoi colpi.

Come funziona

Griglia 10×10 (coordinate x 0–9 da sinistra, y 0–9 dall'alto), flotta 5, 4, 3, 3, 2. Le navi possono toccarsi ma non sovrapporsi. La partita ha due fasi: prima piazzi la flotta, poi si spara a turni alterni (un colpo a testa, anche se fai centro). Vince chi affonda tutto per primo. Chi spara per primo è il Giocatore 1, o il sorteggio nella sfida al meglio di 5.

La busta comune

Ogni messaggio che ricevi contiene anche due campi di busta, omessi negli esempi di questa pagina per brevità: game (qui "navale") e match, un UUID che identifica la singola partita e resta identico dal primo all'ultimo messaggio. All'inizio di ogni partita arriva inoltre un annuncio con la sola busta e tutti i campi di gioco a null: ignoralo, non rispondere. Grazie alla busta un solo main.py può giocare a tutti i giochi, scegliendo il gestore in base a state["game"] — è ciò che rende possibile la modalità torneo.

{"game": "navale", "match": "3f2b1c4e-…", …campi del gioco…}

Fase 1: piazzamento

Ricevi:

{"phase": "place", "grid": {"w": 10, "h": 10}, "fleet": [5, 4, 3, 3, 2],
 "shots": null, "oppShots": null, "winner": null}

Rispondi entro 3 secondi con le tue navi — x, y è la cella di prua (in alto a sinistra), dir è "h" (orizzontale, verso destra) o "v" (verticale, verso il basso):

{"ships": [{"x": 0, "y": 0, "dir": "h", "len": 5},
           {"x": 2, "y": 3, "dir": "v", "len": 4},
           {"x": 5, "y": 5, "dir": "h", "len": 3},
           {"x": 0, "y": 8, "dir": "h", "len": 3},
           {"x": 9, "y": 0, "dir": "v", "len": 2}]}

Piazzamento non valido (lunghezze sbagliate, navi fuori griglia o sovrapposte) = sconfitta immediata.

Fase 2: caccia

Al tuo turno ricevi tutta la storia visibile, così il bot può restare senza stato:

{"phase": "hunt", "grid": {"w": 10, "h": 10}, "fleet": [5, 4, 3, 3, 2],
 "shots": [{"x": 4, "y": 7, "result": "miss"},
           {"x": 5, "y": 2, "result": "hit"},
           {"x": 5, "y": 3, "result": "sunk", "len": 2}],
 "oppShots": [[0, 0], [9, 9]],
 "winner": null}
Campo Significato
shots I colpi che tu hai sparato, in ordine, con l'esito: "miss", "hit" o "sunk" (colpo che affonda: include len, la lunghezza della nave affondata).
oppShots Le coordinate dei colpi ricevuti (gli esiti li conosci: la flotta è tua).

Rispondi entro 2 secondi con le coordinate del colpo:

{"move": [5, 4]}
  • Fuori griglia, cella già colpita da te, tempo scaduto o disconnessione = sconfitta immediata.
  • A fine partita arriva {"phase": null, ..., "winner": "you" | "opp"}: non rispondere.

Esempio completo

import sys
import json
import random


def place_fleet(fleet):
    ships = []
    occupied = set()
    for length in fleet:
        while True:
            d = random.choice(["h", "v"])
            x = random.randrange(11 - length if d == "h" else 10)
            y = random.randrange(11 - length if d == "v" else 10)
            cells = [(x + i, y) if d == "h" else (x, y + i) for i in range(length)]
            if not any(c in occupied for c in cells):
                occupied.update(cells)
                ships.append({"x": x, "y": y, "dir": d, "len": length})
                break
    return ships


def next_shot(shots):
    tried = {(s["x"], s["y"]) for s in shots}
    # mira accanto ai colpi a segno, altrimenti spara a caso
    for s in shots:
        if s["result"] == "hit":
            for dx, dy in ((1, 0), (-1, 0), (0, 1), (0, -1)):
                cell = (s["x"] + dx, s["y"] + dy)
                if 0 <= cell[0] < 10 and 0 <= cell[1] < 10 and cell not in tried:
                    return cell
    while True:
        cell = (random.randrange(10), random.randrange(10))
        if cell not in tried:
            return cell


while True:
    line = sys.stdin.readline()
    try:
        state = json.loads(line)
    except ValueError:
        continue
    if state.get("winner") is not None:
        continue  # partita finita
    if state.get("phase") == "place":
        print(json.dumps({"ships": place_fleet(state["fleet"])}))
    elif state.get("phase") == "hunt":
        x, y = next_shot(state["shots"])
        print(json.dumps({"move": [x, y]}))

Consigli

  • La ricerca a parità di scacchiera (celle con x+y pari) dimezza i colpi a vuoto: la nave più corta è lunga 2.
  • Il vero salto è la mappa di probabilità: per ogni cella, conta in quanti modi le navi non ancora affondate possono passarci, e spara sul massimo.
  • Dopo un "hit", insegui in linea; il "sunk" (con la sua len) ti dice quali hit sono chiusi e quali appartengono a un'altra nave.
  • Anche il piazzamento è strategia: contro le mappe di probabilità, i bordi e gli angoli valgono di più.
  • Per l'emulatore le solite note: CPython, riesecuzione a ogni messaggio — la storia completa nel messaggio esiste apposta.
