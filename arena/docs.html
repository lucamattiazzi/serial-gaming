Arena dei Mostri — API del bot

Sei l'allenatore: scegli la squadra, poi a ogni turno decidi se attaccare o cambiare mostro. Tutto è visibile, niente è casuale: vince chi ragiona meglio.

Come funziona

Si gioca con squadre di 3 mostri scelti da un roster comune di 8. Ogni turno entrambi gli allenatori scelgono simultaneamente: attaccare con una mossa del mostro in campo, o cambiare mostro. I cambi si risolvono prima degli attacchi; poi attacca per primo il mostro più veloce (a parità: colpi simultanei). Un mostro a 0 HP è KO: il suo allenatore deve mandarne in campo un altro. Vince chi azzera la squadra avversaria; dopo 60 turni decide chi ha più HP percentuali. Non c'è nessuna casualità: stesso scenario, stesso risultato.

La busta comune

Come tutti i giochi: ogni messaggio porta game (qui "arena") e match (UUID costante per tutta la partita), e a inizio partita arriva un annuncio con i campi a null da ignorare. Gli esempi omettono la busta per brevità.

I tipi

Quattro tipi in ciclo — fuoco → erba → elettro → acqua → fuoco — più il tipo normale per le mosse. Una mossa fa danno potenza × moltiplicatore:

SituazioneMoltiplicatore
Tipo successivo nel ciclo (es. fuoco → erba)×2 (superefficace)
Tipo precedente nel ciclo (es. fuoco → acqua)×0.5 (resiste)
Stesso tipo×0.5
Tipo «opposto» (fuoco↔elettro, erba↔acqua)×1
Mosse di tipo normalesempre ×1

Fase 1: il draft

Ricevi il roster completo (statistiche e mosse incluse) e rispondi entro 3 secondi con gli indici dei 3 mostri scelti (distinti; l'avversario può scegliere gli stessi):

{"phase": "draft", "roster": [{"name": "Bracino", "type": "fuoco", "maxHp": 80,
  "speed": 95, "moves": [{"name": "Vampata", "type": "fuoco", "power": 40}, ...]}, ...],
 "picks": 3, ...}
{"team": [0, 3, 6]}

Il primo mostro della squadra parte in campo. Squadra non valida = sconfitta.

Fase 2: la battaglia

Al tuo turno ricevi tutto lo stato, aperto (anche la squadra avversaria) più la storia delle azioni — il bot può restare senza stato:

{"phase": "battle", "turn": 4,
 "you": {"active": 0, "team": [{"name": "Bracino", "type": "fuoco", "hp": 42, "maxHp": 80,
          "speed": 95, "moves": [...]}, ...]},
 "opp": {"active": 2, "team": [...]},
 "history": [{"you": ["attacca", 0], "opp": ["cambia", 2]}, ...],
 "winner": null}

Rispondi entro 2 secondi con una delle due azioni:

{"move": ["attacca", 0]}     ← indice della mossa del mostro in campo
{"move": ["cambia", 2]}      ← indice del mostro in squadra (vivo, non già in campo)

Quando il tuo mostro va KO ricevi un messaggio con "phase": "replace": lì è ammesso solo ["cambia", i]. Azione non valida, tempo scaduto o disconnessione = sconfitta. A fine partita: winner = "you"/"opp"/"TIE".

Esempio completo

import sys
import json

CICLO = ["fuoco", "erba", "elettro", "acqua"]


def moltiplicatore(tipo_mossa, tipo_difensore):
    if tipo_mossa == "normale" or tipo_difensore == "normale":
        return 1
    d = (CICLO.index(tipo_difensore) - CICLO.index(tipo_mossa)) % 4
    if d == 1:
        return 2
    if d == 3 or d == 0:
        return 0.5
    return 1


def scegli(state):
    io = state["you"]["team"][state["you"]["active"]]
    lui = state["opp"]["team"][state["opp"]["active"]]
    if state["phase"] == "replace" or io["hp"] <= 0:
        vivi = [i for i, m in enumerate(state["you"]["team"])
                if m["hp"] > 0 and i != state["you"]["active"]]
        return ["cambia", vivi[0]]
    # attacca con la mossa che fa piu' danno
    danni = [m["power"] * moltiplicatore(m["type"], lui["type"])
             for m in io["moves"]]
    return ["attacca", danni.index(max(danni))]


while True:
    line = sys.stdin.readline()
    try:
        state = json.loads(line)
    except ValueError:
        continue
    if state.get("winner") is not None:
        continue  # partita finita
    if state.get("phase") == "draft":
        print(json.dumps({"team": [0, 4, 7]}))  # fuoco + erba + elettro
    elif state.get("phase") in ("battle", "replace"):
        print(json.dumps({"move": scegli(state)}))

Consigli

  • Il cuore è il ciclo dei tipi: attacca il tipo su cui sei superefficace, e cambia mostro quando quello in campo è «preda» dell'avversario.
  • Il cambio costa un turno di danni: fallo solo se il nuovo matchup lo ripaga. E ricorda che l'avversario può prevederlo…
  • Le mosse normali (×1) battono le mosse forti quando queste sarebbero resistite (40 × 0.5 = 20 < 25).
  • Nel draft pensa alla copertura: tre tipi diversi non hanno buchi; due mostri dello stesso tipo cadono con lo stesso avversario.
  • La velocità decide chi colpisce per primo: contro un mostro a un colpo dal KO, essere più veloci vale una vita.
  • Tutto è deterministico: puoi simulare i turni futuri sul RP2040, se ti stanno nei 2 secondi.
