Forza 4 — API del bot

Stesso protocollo del tris, griglia 7×6, gravità e quattro in fila.

Come funziona

Identico al tris: righe JSON sulla seriale USB (115200 baud, 8N1), l'arbitro ti interroga al tuo turno e tu rispondi con la mossa. Sei sempre "O", l'avversario è sempre "X" (ci pensa l'arbitro a rimappare), e puoi muovere per primo — nella sfida al meglio di 5 chi inizia è sorteggiato a ogni partita.

La busta comune

Ogni messaggio che ricevi contiene anche due campi di busta, omessi negli esempi di questa pagina per brevità: game (qui "forza4") e match, un UUID che identifica la singola partita e resta identico dal primo all'ultimo messaggio. All'inizio di ogni partita arriva inoltre un annuncio con la sola busta e tutti i campi di gioco a null: ignoralo, non rispondere. Grazie alla busta un solo main.py può giocare a tutti i giochi, scegliendo il gestore in base a state["game"] — è ciò che rende possibile la modalità torneo.

{"game": "forza4", "match": "3f2b1c4e-…", …campi del gioco…}

Messaggio in ingresso: il tuo turno

{"board": ["", "", ..., "X", "O", "X"], "lastMove": 3, "winner": null}
Campo Significato
board 42 celle ("", "X", "O"), riga 0 in alto, indice = riga * 7 + colonna. La colonna c è piena quando board[c] != "" (la cella in cima).
lastMove La colonna dell'ultima mossa avversaria, null se apri tu.
winner null al tuo turno; "O"/"X"/"TIE" nella notifica di fine partita (a cui non devi rispondere).

La tua risposta

{"move": 3}

L'indice della colonna (0–6) in cui far cadere il tuo disco: finisce nella cella libera più in basso. Le regole:

  • Tempo massimo: 2 secondi. Nessuna risposta = sconfitta.
  • Colonna piena o fuori da 0–6 = sconfitta immediata.
  • Righe non JSON ignorate (comode per il debug via console del browser).
  • Disconnessione durante il tuo turno = sconfitta.

Esempio completo

import sys
import json
import random

COLS = 7


def next_move(board):
    valid = [c for c in range(COLS) if board[c] == ""]
    return random.choice(valid)


while True:
    line = sys.stdin.readline()
    try:
        state = json.loads(line)
    except ValueError:
        continue
    if state.get("winner") is not None:
        continue  # partita finita
    if state.get("board") is None:
        continue  # annuncio di inizio partita
    print(json.dumps({"move": next_move(state["board"])}))

Consigli

  • Forza 4 premia la ricerca: un minimax con potatura alpha-beta e una buona euristica (finestre da 4, preferenza per il centro) domina i bot reattivi. Ma 2 secondi sull'RP2040 sono pochi: misura la profondità che ti puoi permettere.
  • Controlla sempre prima le vittorie immediate tue e le minacce immediate avversarie: costa poco e evita le figuracce.
  • Per l'emulatore valgono le solite note: CPython, programma rieseguito a ogni messaggio, stato persistente solo su file.
